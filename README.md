# VR Holodeck — World Generation System

A VR system that lets users describe a world via voice or text prompt and then generates a walkable 3D environment they can portal into, explore, and return from — all without leaving VR.

See [ROADMAP.md](ROADMAP.md) for full issue descriptions and acceptance criteria.
